# Hundir_la_flota
Proyecto Hundir la Flota

![](https://image.api.playstation.com/cdn/EP0001/CUSA04286_00/kwydaxYoIb3heC7jkdGHTqubYoLHxu46.png)

Proyecto realizado en base a programación Python y Numpy. El objetivo es desarrollar un código que permita realizar una partida imitando el mítico juego de HUndir la Flota. En este caso, el jugador compite con la máquina para ver cuál de los dos es capaz de hundir los barcos en el tablero del enemigo antes.

El juego se inicia con dos tableros, donde se colocan los barcos de forma aleatoria. El jugador va disparando e introduciendo las posiciones de los barcos, a medida que acierta se va completando los huecos en el tablero (x), hasta que se cubren todas las posiciones de los barcos en juego. El jugador se alterna con la máquina para hacer los disparos. Gana el que antes hunda todos los barcos del contrincante.
